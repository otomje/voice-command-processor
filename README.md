# Voice Command Processor
The voice command processor is built on python and vosk models.

## 📚 Project Features
- The project uses the Vosk module in Russian
- One module is used (to save computer resources and optimize speed)
- Opens in a regular terminal
- Works offline

## 📗 What you need to have installed
1. Download the repository as a zip file
2. Python must be installed on your device
3. Open a command prompt, and install these libraries: "pip install vosk sounddevice numpy"
4. Open file assistant.py"
